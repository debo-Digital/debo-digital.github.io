# debo Digital

Official website for debo Digital, hosted via GitHub Pages at [debodigital.github.io](https://debodigital.github.io).

## About

A simple, single-page site featuring the debo Digital name on a clean white background with a footer displaying a copyright notice.

## Structure

- `index.html` — main page markup and styling
- `CNAME` — custom domain configuration for GitHub Pages

## Local Preview

Open `index.html` directly in a browser, or serve it locally:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deployment

This site is automatically published via GitHub Pages from this repository.
